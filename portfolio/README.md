# Kit Portfólio - Aventura RPG

Arquivos entregues:
- `aventura-rpg-card.html`: card para seção de projetos.
- `aventura-rpg-card.css`: estilo do card.
- `aventura-rpg-page.html`: página completa do projeto.

## 1) Adicionar card na seção de projetos
Copie o conteúdo de `aventura-rpg-card.html` para sua seção de projetos e importe o CSS:

```html
<link rel="stylesheet" href="/caminho/aventura-rpg-card.css" />
```

## 2) Ajustar links
Substitua estes placeholders em `aventura-rpg-card.html` e `aventura-rpg-page.html`:
- `https://SEU-USUARIO.github.io/SEU-REPO/web/` -> link da versão web (se existir)
- `https://github.com/SEU-USUARIO/SEU-REPO/releases` -> download
- `https://github.com/SEU-USUARIO/SEU-REPO` -> código

## 3) Se não tiver versão web ainda
No botão "Jogar Agora", você pode:
- apontar para uma página "Em breve", ou
- remover temporariamente esse botão e manter Download + GitHub.

## 4) Prints/GIF para aumentar conversão
Na página `aventura-rpg-page.html`, troque:
- `./assets/prints/gameplay-1.png`
- `./assets/prints/gameplay-2.png`
- `./assets/prints/gameplay-3.png`

por seus prints reais do jogo.

## 5) Texto pronto para card (resumo)
